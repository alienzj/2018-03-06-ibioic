====================
Installing ``BLAST``
====================

We use the sequence search tool `BLAST`_ at several points in the course. This can be
installed as follows:

------------------------
Linux/macOS installation
------------------------

`BLAST`_ is available through `Bioconda`_

.. code-block:: bash

    conda install blast
